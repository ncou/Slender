# Slender example

1. Install composer

      ```text
      $ cd Slender
      $ composer install
      ```

2. Run php server

      ```text
      $ cd Slender
      $ php -S localhost:8888 -t example example/index.php
      ```

3. Open browser

      Open http://localhost:8888 in your browser and you will see 'Welcome to Slender!'
